OCl2-l3 & LEET
LEET is a system of modified spellings originated within the Internet Bulletin Board Systems (BBS) in the 1980s. It uses character replacements to circumvent text filters and censorship.

OCl2-l3 is a modification of the original OCR-B designed by frutiger in the hope of bypassing Optical Character Recognition systems using the language of LEET.
